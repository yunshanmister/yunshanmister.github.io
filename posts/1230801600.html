java开发环境配置

Java
Java
发布日期:   2020-03-26

windows系统安装java

1.下载jdk

首先去oracle官网jdk

下载后JDK的安装根据提示进行，还有安装JDK的时候也会安装JRE，一并安装就可以了。安装JDK，安装过程中可以自定义安装目录等信息。

2.配置环境变量

 1.安装完成后，右击”我的电脑”，点击”属性”，选择”高级系统设置“；

 2.在环境变量窗口下方的系统变量窗口中点击“新建”开始配置。

配置参数：

  • 变量名：JAVA_HOME

  • 变量值：C:\Program Files (x86)\Java\jdk // 要根据自己的实际路径配置“；

  • 变量名：CLASSPATH

  • 变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; //记得前面有个”.”“；

  • 变量名：Path

  • 变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

配置成功后如下

3.测试JDK是否安装成功

1、Win + R，键入”cmd”；

2、键入命令: java -version、java、javac 几个命令，出现以下信息，说明环境变量配置成功；

linux下安装配置环境

1.去官网下载安装包 例如：jdk-8u161-linux-x64.tar.gz 然后上传到

 1.利用tar.gz压缩包来安装，这类软件我们一般都是安装到/opt目录下。依次输入以下命令：

mv jdk-8u161-linux-x64.tar.gz /opt

cd /opt

tar xf jdk-8u161-linux-x64.tar.gz

解压缩完毕后，我们使用ls命令可以看到目录底下又生成了一个新的jdk.18.0_61目录,下一步我们只需要把这个bin目录添加到环境变量里去.就安装配置完成。

 2.配置环境变量：

使用pwd命令查看解压后的文件目录为：/opt/jdk1.8.0_161/bin，我们把这个路径添加到环境变量文件.bash_profile里面去。使用vim编辑器打开这个文件，命令是:

 vim ~/.bash_profile

添加下面的内容

export JAVA_HOME=/home/etl/jdk1.8.0_161
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$JAVA_HOME/bin:$HOME/bin:$HOME/.local/bin:$PATH

保存 （新手请注意： ESC退出 并输入 :wq! 保存并退出） “；

然后执行命令 javac 如未报错误 则配置已成功
